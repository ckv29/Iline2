Система управления персоналом. Руководство.
Это простая система, где можно посмотреть список всех сотрудников, найти нужного и сменить руководителя — прямо на экране.

Для генерации тестовых данных из 50 000 сотрудников выполните команду:
python generate_data.py


Настройки базы должны быть прописаны в файле .env, пример:
DB_USER=postgres
DB_PASSWORD=111
DB_HOST=127.0.0.1
DB_PORT=5432
DB_NAME=iline
SECRET_KEY=2907

Устанавливаем виртуальнное окружение:
virtualenv env
source env/bin/activate

Устанавливаем зависимости:
pip install -r requirements.txt

Запустить сервер ситстемы:
python app.py

Перейди по ссылке:
http://127.0.0.1:5000

Как искать сотрудника
Найти нужного сотрудника по имени (части имени) можно в браузере, вводим в строке URL:
http://127.0.0.1:5000/find_employees/<name>

Как сортировать список
Нажми на одну из кнопок ниже — список отсортируется по выбранному полю:

Сортировать по имени — по алфавиту
Сортировать по должности — например, сначала все CEO
Сортировать по дате — кто пришёл раньше
Сортировать по ЗП — от меньшей к большей
Сортировать по руководителю — кто кому подчиняется


Как сменить руководителя

В поле «Новый руководитель (ID)» введи ID человека, которого хочешь сделать руководителем.
ID — это число (например: 1, 15, 42).
Его можно посмотреть в списке — поле  "Id в базе данных :", также Id руководителя указан в поле руководителя .
Введи id нового руководителя.
Нажми кнопку «Сменить руководителя» рядом с именем сотрудника у которого ты меняешь руководителя.
Готово! Система сразу обновится.

Важно: 
Не вводи свой собственный ID — так нельзя.
Если оставить поле пустым — у сотрудника не будет руководителя.
Если введённого ID нет в системе — появится сообщение об ошибке.
Пример
Хочешь, чтобы Иван Иванов (ID=42) стал руководителем Марии Сидоровой (ID=88):

В поле «Новый руководитель (ID)» введи: 42
Найди Марью Сидорову в списке
Нажми «Сменить руководителя» рядом с её именем
